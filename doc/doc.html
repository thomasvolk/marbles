marbles is a simple documentation tool. Features
- breadcrumb for page hierarchy
- create simple pages witch can reference each other 
- pages can be ordered in hierarchically order
- a page can have a reference to a specific location
- a user can store addresses with location data via handy
- a page can have an association to one location
- it is possible to compose a page with content from different sources
  (svn, delicious, google docs)
